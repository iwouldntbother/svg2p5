# svg2p5

## Simple tool for converting SVG to p5.js code

# WORK IN PROGRESS!!!

Currently it can convert:

- Circle to Ellipse
- Rect to Rect
- Line to Line
- Polygon to Shape (CLOSED)
- Polyline to Shape (OPEN)

To add in the future:

- Path to Shape
- Linear Gradient

Styles that work:

- Fill/noFill
- Stroke/noStroke
- StrokeWeight
- StrokeCap (Square/Round)
